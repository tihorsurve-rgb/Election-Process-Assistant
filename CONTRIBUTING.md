# Contributing to Election Process Assistant

Thank you for your interest in helping improve the Election Process Assistant! We welcome contributions that help make the electoral process more accessible and understandable for everyone.

## Code of Conduct
We are committed to a neutral, factual, and respectful environment. All contributors are expected to uphold these values.

## How to Contribute

### 1. Reporting Bugs
- Use the GitHub Issues tracker to report bugs.
- Include a clear description of the issue and steps to reproduce.

### 2. Suggesting Features
- We welcome suggestions for new features, especially those that leverage Google Services to improve user experience.

### 3. Pull Requests
- Fork the repository.
- Create a new branch for your feature or fix.
- Ensure all tests pass (`npm run test`).
- Submit a pull request with a clear description of your changes.

## Neutrality Guardrails
This project has strict guardrails for the AI Assistant. Any changes to the system prompts must maintain the non-partisan and neutral nature of the assistant.

## Tech Stack
- React + TypeScript
- Google Gemini API
- Tailwind CSS
- Vitest

Happy Coding!
